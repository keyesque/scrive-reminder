# Batch Email Generator

## Instructions
Execute the provided **run.ps1** script to initiate batch reminder.

## Scripts
### Create JSON
Looks through given .xlsx file and saves name, email, and id inside a JSON file for all recipients with specified target color
### Email recipients
Goes through JSON file recipients and sends Scrive reminder mail.
### Run
Runs all scripts in sequential order.

This script is intended for personal use and is tailored to my specific requirements. But if you'd like to use it as a starting off point for writing your own batch email generator feel free to do so :>
